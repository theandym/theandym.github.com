### Selectors


* #### Attribute selectors -

    **Match elements to an attribute that starts with the specified value:**

        [att^="value"] 

    **Match elements to an attribute that ends with the specified value:**

        [att$="value"]

    **Match elements to an attribute that contains the specified value**

        [att*="value"]


    **Example:**

        a[title$="tweetCC"] {
		    position: absolute;
		    top: 0;
		    right: 0;
		    display: block;
		    width: 140px;
		    height: 140px;
		    text-indent: -9999px;
		    }
