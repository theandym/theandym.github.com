webdev_framework
======================

Details
-------

This is a base site framework that implements HTML5 and CSS3. It is intended to
support all modern browsers, including Firefox 3+, Safari 4+, Chrome 5+, and 
Internet Explorer 7+.

Features
--------

* HTML5 Validated
  
* html5doctor.com Reset Stylesheet 1.4.1

* jQuery 1.4.2
